Rigid A Star

==================================================
The Rigid A Star has 5 files.

1) rigid_astar.m is the main file
2) c2g.m is the cost2go function
3) checkposition.m checks clash with obstacles using half planes and semi-algebraic equations
4) new_node.m creates new nodes
5) nodeid.m is used to store nodes as a combination of integers than arrays
6) dist.m is used to calculate parallel lines for minkwoski space

1)RUN rigid_astar.m
2)follow instructions
3)output will be displayed
